**Title:** Development of an application for monitoring and managing load and traffic balancing for web services
(DNS-based Global Server Load Balancing, DNS GSLB).

**Author:** Aleksejs Čudovs  
**Supervisor:** Boriss Mišņevs  
**Year:** 2016

---

## Contents

The thesis covers two parts:

1. **Survey:** load balancing at the physical layer, data link layer (LAG, MC-LAG), and network layer (ECMP, RIP, OSPF,
   EIGRP, BGP); server load balancing (NAT, direct routing, tunneling); global server load balancing (HTTP redirect,
   DNS, routing protocols); comparison of open-source technologies.

2. **Application development:** requirements, architecture, database design, implementation, testing, and quality
   assessment of what became PowerGSLB.
